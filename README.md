# 📝 To-Do App

A simple To-Do App built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Task counter (Total, Done, Pending)
- Dark mode
- Data saved using localStorage

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
